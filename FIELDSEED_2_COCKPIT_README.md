# FieldSeed 2.0 Cockpit Update

This build turns FieldSeed into a desktop-first command cockpit using CustomTkinter.

## Launch

Double-click:

```text
START_FIELDSEED.bat
```

or run:

```powershell
python -m pip install customtkinter
python -m fieldseed app
```

## What changed

- Modern CustomTkinter desktop shell
- Persistent right-side FieldSeed Brain panel
- Mission Control landing page
- Editable Intake review workflow
- Manual tickets can be parsed through Intake too
- Observe Mode screen
- Arsenal / tool hub
- Safety, Brain, Memory, USB Rescue, Growth screens

## Important

The CLI still exists for automation and USB recovery, but the intended daily experience is now the desktop app.
